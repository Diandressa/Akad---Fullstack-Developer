## Programação síncrona vs. assíncrona

* Programação síncrona executa tarefas sequencialmente, bloqueando o código até que cada operação termine. 
* Programação assíncrona permite iniciar tarefas sem esperar a conclusão das anteriores, liberando o sistema para processar outras instruções simultaneamente. O síncrono é mais simples, enquanto o assíncrono melhora a performance e a fluidez
